## ✨ Overview

The **Python 30-Day Mastery Tracker** is a **Progressive Web App (PWA)** designed to help you systematically master Python in 30 days. This isn't just a website—it's a full-featured app that works offline, sends notifications, and feels like a native mobile application.

Transform your Python learning journey with an interactive, gamified experience that tracks progress, provides curated resources, and keeps you motivated every step of the way.

---

## 🚀 Key Features

### 📱 **Progressive Web App (PWA)**
- **Installable:** Add to your phone's home screen like a native app
- **Offline Support:** Continue learning even without internet connection
- **Push Notifications:** Daily reminders to keep you on track
- **Background Sync:** Progress syncs automatically when connection returns
- **Native App Feel:** Splash screen, app icons, and smooth animations

### 🎯 **Learning Features**
- **30-Day Structured Curriculum:** From basics to advanced topics
- **Interactive Progress Tracking:** Visual progress ring and achievement system
- **Curated Resource Links:** Direct access to quality tutorials and documentation
- **Daily Task Completion:** Mark morning and evening sessions
- **Streak Tracking:** Build momentum with consecutive day counters
- **Achievement System:** Unlock badges as you progress

### 🎨 **Modern UI/UX**
- **Glassmorphism Design:** Beautiful translucent interface with blur effects
- **Micro-animations:** Smooth transitions and hover effects
- **Haptic Feedback:** Tactile responses on mobile devices
- **Responsive Design:** Perfect on desktop, tablet, and mobile
- **Accessibility:** Screen reader friendly and keyboard navigable

## 💻 Local Development

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Web server (for full PWA features)

### **Setup**
1. **Clone the repository:**
   ```bash
   git clone https://github.com/jb24000/PythonMasteryTracker.git
   cd PythonMasteryTracker
   ```

2. **Serve locally:**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser:**
   ```
   http://localhost:8000
   ```

### **File Structure**
```
PythonMasteryTracker/
├── index.html          # Main application
├── manifest.json       # PWA configuration
├── sw.js              # Service worker for offline support
└── README.md          # This file
```

## 🛠️ Technical Features

### **PWA Capabilities**
- **Service Worker:** Caches resources for offline use
- **Web App Manifest:** Defines app metadata and appearance
- **Background Sync:** Syncs data when connection is restored
- **Push API:** Sends daily learning reminders
- **Cache API:** Stores educational resources offline
- **IndexedDB:** Persistent local data storage

## 📚 Curriculum Overview

**Week 1:** Python Fundamentals
- Days 1-2: Setup, Hello World, Variables
- Days 3-5: Data Types, Lists, Dictionaries
- Days 6-7: Control Flow, Loops

**Week 2:** Intermediate Concepts  
- Days 8-10: Functions, Arguments, Error Handling
- Days 11-14: File I/O, List Comprehensions, Modules

**Week 3:** Object-Oriented Programming
- Days 15-18: Classes, Objects, Inheritance, Polymorphism
- Days 19-21: Advanced Functions, Virtual Environments

**Week 4:** Real-World Applications
- Days 22-26: Web Scraping, APIs, Data Analysis, Visualization
- Days 27-30: Testing, Web Apps, Deployment, Capstone Project
